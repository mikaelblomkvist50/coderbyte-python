# Return the Sum of Two Numbers

Create a function that takes two numbers as arguments and `return` their sum.

# Examples

```
3, 2 ➞ 5

-3, -6 ➞ -9

7, 3 ➞ 10
```
# Notes

* Don't forget to return the result.
